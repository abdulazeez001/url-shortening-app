# URL Shortening Application

This application is built using AdonisJS, providing functionalities for user registration, login, logout, and link shortening. It also includes authentication and validation using Vine.

### User Logout

- **URL:** `/api/user/logout`
- **Method:** `POST`
- **Description:** Logs out the authenticated user. Requires authentication.

### Get Links

- **URL:** `/api/links?page=1&limit=10`
- **Method:** `GET`
- **Description:** Retrieves all links for the authenticated user page is 1 and limit is 10 by default. Requires authentication.
